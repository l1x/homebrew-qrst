# homebrew-qrst

Homebrew tap for [qrst](https://github.com/l1x/qrst) — on-device hybrid search for markdown files.

## Install

```bash
brew tap l1x/qrst
brew install qrst
```

## Usage

qrst requires an ONNX embedding model. See the [qrst README](https://github.com/l1x/qrst) for setup instructions.
